TemplateRef and ngTemplateOutlet: blog (static)

This app shows static blog entries.   (Branch 15 is the same with a form to add new entries.)

default

To show the default content, app-blog-list should be empty.

default with addition

app-blog-list contains ng-template #additionTemplate.
Its content addition of type TemplateRef<any> is bound to ngTemplateOutlet.
It is styled with ::ng-deep in blog-list.component.css. (Same in the following examples.)

custom minimal

app-blog-list contains ng-template #customTemplate.
Its minimal content is shown once for each blog entry.

custom

Like the last example, but here the count variable entry is defined using let-entry, so {{entry.title}} can be used in the content.

custom fail

The custom template uses table rows. But they would require a table in place of the div.blog-list-custom.
The next branch shows an attempt to solve this with ng-container, but that does not work either.
Another problem is, that the addition can not be shown below the table.
